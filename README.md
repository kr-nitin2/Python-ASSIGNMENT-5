# Python-ASSIGNMENT-5
Task-1

Problem Statement : Create a dictionary where students names are keys and their marks are values, let user search the marks of the student.
Functionality:
- A dictionary is created having names and the marks of the student.
- The user inputs the name to search.
- If the name exists in the dictionary then it displays the marks of the student.
- If not then the program displays "Student not found".
